# AI 지시 기록

계약 변경, Core 산출 규칙, 평가 기준처럼 재현이 필요한 지시만 이 폴더에 남긴다. 일회성 대화나 API 키·토큰·개인정보는 저장하지 않는다.

파일명은 `<track>_<topic>.md` 형식을 사용한다. `track`은 `product`, `design`, `engineering`, `cross-review` 중 하나다.

```markdown
# 작업명

- 작성일:
- 담당 트랙:
- 기준 계약 버전:
- 입력 문서:
- 허용 산출물:
- 금지 범위:
- 완료 기준:
- 실행한 검증:
- 결과 artifact:
```

산출물이 계약 변경을 요구하면 지시 기록만 고치지 말고 [`../contracts/`](../contracts/)와 영향 문서를 같은 변경 묶음에서 갱신한다.
