# 타슈캐스트 문서

현재 구현 기준은 [`v2/README.md`](./v2/README.md)에서 시작한다.

## 현재 기준

- `v2/01_PRODUCT_MVP.md`: 제품 범위와 UX
- `v2/02_DATA_AND_EVALUATION.md`: 데이터 검증과 평가
- `v2/03_ARCHITECTURE_SECURITY.md`: 아키텍처·보안
- `v2/04_EXECUTION_ROADMAP.md`: 실행 순서
- `v2/05_OPERATING_MODEL.md`: 사람×AI 페어 3트랙 운영 방식
- `v2/contracts/`: OpenAPI, enum, 불변 규칙
- `v2/prompts/`: 재현이 필요한 AI 지시 기록 형식

## 보존 문서

- `PLAN.md`: 기존 2일 해커톤 통합안
- `PLAN_AI.md`: 기한 무제약 AI 수행 탐색안
- `TEAM_GUIDE.md`: 기존 사람팀 역할·API 초안
- `research/R1~R4`: 조사 자료
- `research/D1~D3`: 보존된 섹션 초안
- `research/V1`: 과거 정합성 감사 기록

보존 문서와 v2가 충돌하면 v2 계약을 따른다. 단, 실제 API fixture 또는 재현 가능한 테스트가 문서와 충돌할 경우에는 계약과 영향 문서를 함께 갱신한다.
